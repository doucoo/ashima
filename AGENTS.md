# AGENTS

## Highest Priority Project-Start Rule

Before starting any future new project inside the `ashima` repository, you MUST first read all three of these documents in this repository:

1. `docs/global-workbench.md`
2. `docs/global-compounding-and-pitfalls-log.md`
3. `docs/new-project-sop.md`

You may only proceed with a new project after confirming those three documents have been read and their rules will be followed.

## Scope Rule

The rules above currently apply inside the `ashima` repository only. Do not automatically propagate them to other repositories until the workflow has matured here and the user explicitly asks for promotion.
